# 废弃路由文档归档

## 概述

此目录包含项目早期版本中已废弃的路由和相关代码文件，仅供历史参考和迁移记录使用。

## 目录结构

```
deprecated-routes/
├── index.js              # 旧版主服务文件
└── routes/               # 旧版路由文件
    ├── platforms.js      # 平台管理路由
    ├── publish.js        # 发布管理路由
    └── tasksGo.js        # Go任务代理路由
```

## 废弃原因

这些文件已被新的Go后端服务替代，主要原因包括：

1. **架构升级**: 从Node.js迁移到Go语言实现
2. **性能优化**: Go在并发处理方面表现更优
3. **维护简化**: 统一后端技术栈
4. **功能整合**: 将分散的功能整合到publisher-core中

## 历史版本信息

### 最后使用版本
- **版本**: v0.9.x
- **停用时间**: 2026年1月
- **替代方案**: publisher-core Go服务

### 主要差异对比

| 功能 | 旧版Node.js实现 | 新版Go实现 |
|------|----------------|------------|
| 平台管理 | `routes/platforms.js` | `publisher-core/adapters/` |
| 内容发布 | `routes/publish.js` | `publisher-core/api/` |
| 任务管理 | `routes/tasksGo.js` | `publisher-core/task/` |
| 浏览器自动化 | Puppeteer | Rod框架 |
